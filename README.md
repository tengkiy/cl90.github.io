# SCL-90症状自评量表

一个独立的、无需后端的SCL-90心理测评页面，适合在微信中分享使用。

## 功能特点

- ✅ 完整的SCL-90量表（90道题目）
- ✅ 授权码验证机制
- ✅ 每个授权码每天只能使用一次
- ✅ 无需数据库，使用localStorage存储
- ✅ 自动计算总分和9个因子分
- ✅ 提供个性化结果建议
- ✅ 响应式设计，适配手机端
- ✅ 无需服务器，纯前端实现

## 使用方法

### 1. 部署
将以下文件上传到任何静态网站托管服务：
- `index.html`
- `scl90.js`

推荐的免费托管平台：
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages

### 2. 修改授权码
在 `scl90.js` 文件中找到 `authCodes` 数组，修改为你自己的授权码：

```javascript
const authCodes = [
    "YOUR_CODE_1",
    "YOUR_CODE_2",
    "YOUR_CODE_3",
    // 添加更多授权码...
];
```

### 3. 分享
将部署后的网址分享给需要测评的人，并告知授权码。

## 授权码机制

- 每个授权码每天只能使用一次
- 使用记录存储在浏览器的localStorage中
- 每天自动清理过期记录
- 可以预设多个授权码供不同用户使用

## 评分说明

### 总分解读
- < 160分：正常范围
- 160-200分：轻度症状
- > 200分：需要专业关注

### 因子分解读
- < 2分：正常
- 2-2.5分：轻度
- > 2.5分：需要关注

### 9个因子
1. 躯体化
2. 强迫症状
3. 人际关系敏感
4. 抑郁
5. 焦虑
6. 敌对
7. 恐怖
8. 偏执
9. 精神病性

## 技术栈

- 纯HTML + CSS + JavaScript
- 无需后端服务器
- 使用localStorage进行本地存储
- 响应式设计

## 隐私保护

- 所有数据仅存储在用户浏览器本地
- 不会上传任何测评数据到服务器
- 清除浏览器数据即可删除所有记录
